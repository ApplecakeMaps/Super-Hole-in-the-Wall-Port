# Play the classic Super Hole in the Wall! map in the modern versions of Minecraft! (26.1.2)
This ported version of Super Hole in the Wall! is a fan creation and is not affiliated nor endorsed by The Noxcrew or Avondale.
This map maintains a majority of the original code with updates to syntax as well as additional changes to maintain similar functionality and appearance to the original.
